Module Command

module Command: sig .. end

Parses user commands

type index = int

The type index represents the index of a card in a player's hand.

type card_triple = index * index * index

The type card_triple represents the index of three unique cards in a player's hand. Thus, no index in a card_triple may be the same.

type command =
| Quit
| Pass of card_triple
| Play of index
| Select of string
| Help
| Restart
| Back
| Deal
| Debug

The type command represents a command inputted by the player. It is composed of a verb and optional index arguments or difficulty string.

exception Empty

Raised when the user inputs an empty command.

exception Malformed

Raised when the user inputs a malformed command.

exception InvalidIndex

Raised when the user inputs an index outside of the range of indices in the player's hand of cards.

val get_first : string list -> string

get_first lst is the first word in lst. If lst is empty, raises Empty exception.

val get_rest : 'a list -> 'a list

get_rest lst is the lst with the first word removed. If lst is empty, raises Empty exception.

val get_words : string list -> string list -> string list

get_words acc lst is all the words in lst, which does not include the empty string.

val get_index : string -> int

get_index ind is the integer represented by ind. If ind does not represent a valid integer, then raises Malformed.

val make_play : string list -> command

make_play rest is the Play command with the index of the card to play in rest. If rest does not contain one integer index, then raises Malformed.

val make_select : string list -> command

make_select rest is the Select command with the corresponding difficult contained in rest. If rest does not contain a single word representing a difficulty, then raises Malformed.

val make_pass : string list -> command

make_pass rest is the Pass command with the indices of the three cards to pass in rest. If rest does not contain three valid indices, then a Malformed exception is raised.

val make_command : string list -> string -> command

make_command rest lead is the Command represented by the lead word lead and the command phrase in rest. If these do not correspond to a valid command, then raises Malformed.

val parse : string -> command

parse str is the command corresponding to the input entered by the user str. Raises Malformed if str does not represent a valid command. That is, if str does not contain a valid combination of command verbs and optional index arguments or difficulty string argument. If str is empty, raises Empty.
